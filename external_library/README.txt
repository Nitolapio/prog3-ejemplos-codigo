Para generar la librería utilizada en los ejemplos del cap03:

1. Entrar en el directorio external_library: cd external_library
2. Compilar todas las clases del directorio es.deusto.prog3: javac es/deusto/prog3/*.java
3. Crear un fichero .zip que contenga los directorios que hay dentro del directorio external_library.
    Lo importante es que incluya los .class generados, el código fuente no es necesario.
4. Cambiar el nombre al fichero .zip creado a example-lib.jar
